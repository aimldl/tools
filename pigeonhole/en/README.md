# 피젼홀 라이브 (Pigeonhole Live)

* 피젼홀 라이브는 온라인 Q&A 플랫폼입니다.
* 피젼홀을 통해 온라인으로 실시간/사전 질문을 받고, 투표결과를 손쉽게 모을 수 있습니다.

* https://pigeonholelive.com/
* https://www.pigeonhole.at/
