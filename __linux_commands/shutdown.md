# shutdown

## Halt the system
```bash
$ sudo shutdown -h now
```

## Reboot the system
For details, refer to [Reboot Linux System Command](https://www.cyberciti.biz/faq/howto-reboot-linux/).
```bash
$ sudo shutdown -r now
```
or
```bash
$ sudo reboot
```
