#### Syntax
```bash
$ nano <file_name>
```
| Shortcut | Menu                |
|----------|---------------------|
| alt+g    | Go to line          |
| ctrl+x   | Exit                |
| ctrl+o   | write Out (or save) |

##### -l or	--linenumbers
Show line numbers in front of the text.

```bash
$ nano -l 136 <file_name>
```
