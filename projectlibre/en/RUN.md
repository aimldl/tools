## How to Run `projectlibre`

### Run `projectlibre` on terminal

1. Open a terminal or press `Ctrl+alt+t`
2. Run

```bash
$ projectlibre &
```

### Run `projectlibre` on GUI

1. Go to `Show Applications`

2. Search and launch projectlibre as follows.
