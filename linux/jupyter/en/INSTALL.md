## Jupyter Notebook

## 설치하기

아나콘다를 설치하면 자동으로 주피터 노트북도 같이 설치됩니다.

## 실행하기

터미널에 아래 명령어를 입력하면 주피터 노트북이 열립니다. 

```bash
(base) $ jupyter notebook
```
